# Estoque Livros AC - Frontend

Frontend React para o sistema de gestão de estoque de livros.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em http://localhost:3000

## Build

```bash
npm run build
```

## Estrutura do Projeto

- `src/pages/` - Páginas principais (Login, Dashboard, Movimentações)
- `src/api/` - Integração com API backend
- `src/store/` - State management com Zustand
